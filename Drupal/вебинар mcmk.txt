1. В админке открываем "Содержимое" - "Статический вебинар"
2. В админке открываем "Содержимое" - "Добавить материал" - "Тренинг"
3. Из "Статический вебинар" в "Тренинг" копируем

Тема вебинара -> Название тренинга
Тема вебинара -> Тема вебинара
Категория тренинга/занятия
Дата и время проведения
Видео с вебинаром
Лектор
Описание тренинга

Сохраняем

4. В "Статический вебинар" меняем:

Тема вебинара (с баннера, который прислали)
Дата и время проведения
Отмечаем "Заглушка вместо ролика"
Подгружаем картинку с баннером
Из "Видео с вебинаром" убираем всё содержимое
Описание тренинга
	CTRL + C CTRL + V всё, что до цитаты с ссылкой
	CTRL + C CTRL + V Текст цитаты
	Удалить старую ссылку, вставить новую
	CTRL + C CTRL + V текст после цитаты с ссылкой
	CTRL + C CTRL + V Текст второй ссылки 
	Удалить старую ссылку, вставить новую

5. Сохраняемся и сбрасываем кэш
6. проверяем с инкогнито
